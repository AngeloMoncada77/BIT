Sobre mí

Desarrollador web, creador de herramientas y estudiante eterno de la tecnología.

Hola, soy AMQ7 👋

Soy un desarrollador web apasionado por construir cosas que funcionen, se vean bien y ayuden a otros. Mi enfoque está en el desarrollo frontend con JavaScript puro, HTML semántico y CSS moderno, creando aplicaciones web rápidas, accesibles y sin dependencias innecesarias.

Creo firmemente que la mejor manera de aprender tecnología es combinando tres cosas: leer sobre cómo funcionan las cosas, construir herramientas que resuelvan problemas reales, y compartir el conocimiento con otros. Este sitio web es la materialización de esa filosofía.

¿Qué hago?

Desarrollo aplicaciones web del lado del cliente utilizando HTML, CSS y JavaScript. Me interesa especialmente la creación de herramientas educativas gratuitas, la divulgación técnica sobre redes y programación, y la exploración de la arquitectura detrás de los videojuegos multiplayer.

Cada proyecto en mi portafolio está diseñado para ser útil por sí mismo — no son demos vacíos, sino herramientas que cualquier persona puede usar directamente desde el navegador.

Tecnologías que uso

HTML5 CSS3 JavaScript Git Netlify VS Code Responsive Design SEO

Más allá del código

Cuando no estoy programando, me encontrarás explorando temas de redes y gaming, leyendo sobre arquitectura de sistemas, o buscando formas de hacer que la tecnología sea más accesible para quienes están empezando.

Este sitio es un proyecto en evolución constante. Siempre hay algo nuevo que aprender, algo que mejorar y algo que compartir.
