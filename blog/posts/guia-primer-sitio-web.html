Guía para principiantes: cómo crear y desplegar tu primer sitio web gratis

Julio 2026 · Programación

Crear un sitio web desde cero puede parecer abrumador cuando no sabes por dónde empezar. Pero la realidad es que hoy puedes tener tu propio sitio en Internet completamente gratis en menos de una hora. Esta guía te lleva paso a paso sin asumir ningún conocimiento previo.

Paso 1: Prepara tu entorno de trabajo

Necesitas dos herramientas básicas:

  • Visual Studio Code — editor de código gratuito. Descárgalo de code.visualstudio.com
  • Live Server — extensión de VS Code que te permite ver los cambios en tiempo real. Se instala desde la pestaña de extensiones (Ctrl+Shift+X).

Una vez instalado VS Code, crea una carpeta en tu computadora llamada mi-sitio y ábrela con VS Code (Archivo → Abrir carpeta).

Paso 2: Crea tu primer archivo HTML

Dentro de VS Code, crea un archivo llamado index.html y escribe esto:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Mi primera página</title>
</head>
<body>
  <h1>¡Hola, mundo!</h1>
  <p>Esta es mi primera página web.</p>
</body>
</html>

Haz clic derecho sobre el archivo y selecciona "Open with Live Server". Se abrirá tu navegador mostrando tu página. Cambia el texto y guarda — verás cómo se actualiza automáticamente.

Paso 3: Agrega CSS para dar estilo

Crea una carpeta llamada css y dentro un archivo style.css:

body {
  font-family: system-ui, sans-serif;
  background: #0f172a;
  color: #e2e8f0;
  max-width: 600px;
  margin: 0 auto;
  padding: 2rem;
  text-align: center;
}
h1 { color: #38bdf8; }

Enlázalo desde tu HTML añadiendo esta línea dentro de <head>:

<link rel="stylesheet" href="css/style.css">

Paso 4: Agrega JavaScript para interactividad

Crea una carpeta js y dentro un archivo script.js:

document.querySelector('h1').addEventListener('click', () => {
  alert('¡Hiciste clic en el título!');
});

Enlázalo antes del cierre de </body>:

<script src="js/script.js"></script>

Paso 5: Sube tu código a GitHub

GitHub es donde guardarás tu código en la nube. Crea una cuenta gratuita en github.com.

  1. En GitHub, haz clic en el icono + → New repository
  2. Nómbralo mi-sitio y haz clic en Create repository
  3. En tu terminal de VS Code (Ctrl+Ñ), ejecuta: 
    git init
git add -A
git commit -m "Primer commit"
git branch -M main
git remote add origin https://github.com/tu-usuario/mi-sitio.git
git push -u origin main

Paso 6: Despliega tu sitio en Netlify (gratis)

  1. Crea una cuenta en netlify.com (puedes usar tu cuenta de GitHub)
  2. Haz clic en Sites → Import from Git → GitHub
  3. Selecciona tu repositorio mi-sitio y haz clic en Deploy

En menos de un minuto, Netlify te dará una URL pública como mi-sitio.netlify.app. ¡Tu sitio ya está en Internet!

Paso 7: Conecta un dominio personalizado

Puedes comprar un dominio como tunombre.xyz por unos 2-3 dólares al año. En Netlify ve a Site settings → Domain management → Add custom domain y sigue las instrucciones para configurar los DNS.

Conclusión

En menos de una hora has creado un sitio web, lo has subido a GitHub y lo has desplegado en Internet completamente gratis. A partir de aquí, puedes añadir más páginas, mejorar los estilos y agregar funcionalidades con JavaScript.

El truco está en empezar simple. No intentes construir el sitio perfecto de golpe. Crea algo pequeño, publícalo, y luego mejóralo. Cada sitio web profesional que ves empezó exactamente así.
