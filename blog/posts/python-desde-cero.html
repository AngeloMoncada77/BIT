Python desde cero: Todo lo que necesitas saber para empezar en 2026

Python es el lenguaje de programación que más ha crecido en los últimos años, y en 2026 sigue siendo la puerta de entrada ideal para quienes quieren iniciarse en el mundo del desarrollo. Desde inteligencia artificial y ciencia de datos hasta automatización de tareas y desarrollo web, Python está en todas partes. En esta guía recorreremos desde la instalación hasta tus primeras funciones, sin asumir ningún conocimiento previo.

¿Por qué Python en 2026?

Python se ha consolidado como el lenguaje favorito en áreas clave de la tecnología:

  • Inteligencia Artificial y Machine Learning: Frameworks como TensorFlow, PyTorch y scikit-learn tienen a Python como su interfaz principal.
  • Ciencia de Datos: Con librerías como pandas, NumPy y Matplotlib, Python es la herramienta estándar para análisis de datos.
  • Desarrollo Web: Django y Flask permiten crear aplicaciones web robustas y escalables.
  • Automatización y Scripting: Python es perfecto para automatizar tareas repetitivas, procesar archivos y trabajar con APIs.
  • Comunidad y aprendizaje: Su sintaxis clara y legible lo convierte en el mejor lenguaje para aprender a programar.

Instalación de Python

Antes de escribir código, necesitas tener Python instalado en tu equipo. Sigue los pasos según tu sistema operativo.

En Windows

Ve a python.org, descarga la última versión estable (Python 3.13 o superior) y ejecuta el instalador. Importante: marca la casilla "Add Python to PATH" antes de hacer clic en "Install Now".

En macOS

Descarga el instalador desde python.org o instálalo con Homebrew:

brew install python3

En Linux (Ubuntu/Debian)

Python suele venir preinstalado, pero puedes asegurarte de tener la versión más reciente:

sudo apt update
sudo apt install python3 python3-pip

Verificar la instalación

Abre una terminal o consola y ejecuta:

python --version
pip --version

Si ves los números de versión (por ejemplo Python 3.13.2), todo está listo.

Tu primer programa: Hola Mundo

Crea un archivo llamado hola.py y escribe lo siguiente:

print("¡Hola, Mundo!")

Guarda el archivo y ejecútalo desde la terminal:

python hola.py

Deberías ver el mensaje ¡Hola, Mundo! en pantalla.

Explicación línea por línea

print() es una función incorporada de Python que muestra texto en la consola. Todo lo que va entre paréntesis se evalúa y se imprime. Las comillas dobles " " indican que el contenido es una cadena de texto (string).

Variables y tipos de datos

En Python no necesitas declarar el tipo de una variable; el intérprete lo infiere automáticamente.

nombre = "Ana"
edad = 25
altura = 1.68
es_estudiante = True

print(type(nombre))   # <class 'str'>
print(type(edad))     # <class 'int'>
print(type(altura))   # <class 'float'>
print(type(es_estudiante))  # <class 'bool'>

Los tipos básicos en Python son:

  • int: números enteros (ej: 25, -3, 0)
  • float: números decimales (ej: 3.14, 1.68)
  • str: cadenas de texto (ej: "Hola", 'Python')
  • bool: valores booleanos (True o False)

Usa la función type() para conocer el tipo de cualquier variable.

Strings

Las cadenas de texto son uno de los tipos más usados. Python ofrece múltiples formas de trabajar con ellas.

Concatenación

saludo = "Hola, " + "mundo"
print(saludo)  # Hola, mundo

f-strings (recomendado)

nombre = "Carlos"
edad = 30
mensaje = f"Me llamo {nombre} y tengo {edad} años."
print(mensaje)  # Me llamo Carlos y tengo 30 años.

Métodos básicos

texto = "  Python es INCREÍBLE  "

print(texto.upper())       # "  PYTHON ES INCREÍBLE  "
print(texto.lower())       # "  python es increíble  "
print(texto.strip())       # "Python es INCREÍBLE"
print(texto.split())       # ['Python', 'es', 'INCREÍBLE']

Listas y diccionarios

Listas

Una lista es una colección ordenada y mutable de elementos.

frutas = ["manzana", "banana", "cereza"]
print(frutas[0])      # manzana

frutas.append("durazno")
print(frutas)         # ['manzana', 'banana', 'cereza', 'durazno']

frutas.pop()          # Elimina el último elemento
print(frutas)         # ['manzana', 'banana', 'cereza']

Diccionarios

Un diccionario almacena pares clave-valor.

persona = {
    "nombre": "Lucía",
    "edad": 28,
    "ciudad": "Bogotá"
}

print(persona["nombre"])       # Lucía
print(persona.keys())          # dict_keys(['nombre', 'edad', 'ciudad'])
print(persona.values())        # dict_values(['Lucía', 28, 'Bogotá'])

Condicionales

Los condicionales permiten ejecutar código según una condición.

edad = 18

if edad >= 18:
    print("Eres mayor de edad")
elif edad >= 13:
    print("Eres adolescente")
else:
    print("Eres menor de edad")

Presta atención a los : al final de cada condición y a la indentación (4 espacios). En Python, la indentación es obligatoria y define los bloques de código.

Bucles

Bucle for

# Iterar sobre una lista
colores = ["rojo", "verde", "azul"]
for color in colores:
    print(color)

# Usar range()
for i in range(5):
    print(i)  # 0, 1, 2, 3, 4

for i in range(2, 8, 2):
    print(i)  # 2, 4, 6

Bucle while

contador = 0
while contador < 5:
    print(f"Vuelta {contador}")
    contador += 1

Funciones

Las funciones permiten reutilizar bloques de código. Se definen con def.

def saludar(nombre):
    return f"¡Hola, {nombre}!"

mensaje = saludar("María")
print(mensaje)  # ¡Hola, María!

Función con varios parámetros

def sumar(a, b):
    return a + b

resultado = sumar(10, 5)
print(resultado)  # 15

La palabra clave return devuelve un valor desde la función. Si no usas return, la función devuelve None.

Próximos pasos

Has cubierto los fundamentos de Python. Ahora te recomendamos:

  • Practicar con proyectos pequeños: Calculadora, generador de contraseñas, juego de adivinanzas.
  • Explorar librerías: random, datetime, os para empezar.
  • Apuntar a un área específica: Si te gusta la web, prueba Flask. Si te atrae la ciencia de datos, aprende pandas.
  • Usar plataformas interactivas: LeetCode, Exercism o Project Euler para resolver ejercicios.
  • Leer documentación oficial: docs.python.org es tu mejor aliada.

Conclusión

Python es un lenguaje accesible, versátil y con una comunidad enorme. En esta guía viste desde la instalación hasta funciones, pasando por variables, strings, listas, diccionarios, condicionales y bucles. La clave ahora es la práctica constante. Escribe código todos los días, aunque sean solo unos minutos, y verás cómo tu comprensión crece rápidamente. ¡El momento de empezar es ahora!
