AI Study Buddy

Generador de flashcards inteligentes. Pega tus apuntes o textos y la herramienta extrae automáticamente conceptos clave, generando tarjetas de preguntas y respuestas para estudiar de forma interactiva. Ideal para estudiantes que quieren repasar rápido antes de un examen.

Procesa el texto ingresado, identifica términos importantes usando palabras clave y patrones, y los convierte en un mazo de flashcards navegables. Incluye control de progreso y modo aleatorio.

JavaScript Productividad

Dev Task Dashboard

Tablero de tareas estilo Kanban para gestionar proyectos personales. Organiza tus pendientes en columnas de "Por hacer", "En progreso" y "Completado" con arrastrar y soltar nativo.

Los datos se guardan automáticamente en LocalStorage, por lo que el estado persiste incluso después de cerrar el navegador. Incluye búsqueda de tareas y contador de progreso visual.

Markdown Editor

Editor de Markdown en vivo con vista previa instantánea y exportación a HTML. Escribe en el panel izquierdo y ve el resultado renderizado en tiempo real en el panel derecho.

Soporta encabezados, listas, enlaces, imágenes, código, tablas y más. Incluye botón de copia del HTML generado para usar en blogs o plataformas educativas.
