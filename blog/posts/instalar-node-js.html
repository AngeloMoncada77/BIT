Cómo instalar Node.js en Windows, Linux y Mac — Guía completa 2026

Node.js se ha convertido en una herramienta esencial para cualquier desarrollador web. Si estás empezando o cambiaste de equipo, aquí tienes una guía paso a paso para instalarlo en los tres sistemas operativos principales.

¿Qué es Node.js?

Node.js es un entorno de ejecución para JavaScript fuera del navegador. Te permite usar JS para crear servidores, automatizar tareas, construir herramientas de línea de comandos y mucho más. Viene acompañado de npm (Node Package Manager), el gestor de paquetes más grande del mundo.

Método recomendado: instalar con nvm

La forma más flexible de instalar Node.js es usando nvm (Node Version Manager). Esto te permite instalar y cambiar entre múltiples versiones de Node fácilmente.

En Linux / macOS

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.0/install.sh | bash

Después de la instalación, cierra y vuelve a abrir la terminal, o ejecuta:

source ~/.bashrc

Luego instala la versión más reciente de Node.js:

nvm install node  # Última versión
nvm install 20     # Versión específica (LTS)

En Windows

En Windows puedes usar nvm-windows. Descárgalo desde nvm-windows releases e instala el archivo nvm-setup.exe. Luego abre una terminal como administrador:

nvm install latest
nvm use latest

Instalación directa desde nodejs.org

Si prefieres no usar nvm, ve a nodejs.org y descarga el instalador de tu sistema operativo:

  • Windows: Descarga el archivo .msi y ejecútalo. Marca la opción "Add to PATH".
  • macOS: Descarga el archivo .pkg e instálalo.
  • Linux: Usa el gestor de paquetes de tu distribución o los binarios precompilados.

Instalación en Ubuntu/Debian

curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt-get install -y nodejs

Verificar la instalación

Abra una terminal y ejecuta:

node --version
npm --version

Si ves los números de versión, todo está funcionando correctamente.

Primeros pasos con Node.js

Crea un archivo hola.js y escribe:

console.log("¡Hola, Node.js!");

Ejecútalo:

node hola.js

Si ves el mensaje, Node.js funciona correctamente.

Solución de problemas comunes

  • "node no se reconoce como comando": No agregaste Node al PATH. Reinstala marcando la opción o agrega la ruta manualmente.
  • Error de permisos en Linux/macOS: Usa sudo para la instalación global o mejor usa nvm.
  • nvm command not found: Asegúrate de haber reiniciado la terminal o ejecutado source ~/.bashrc.

Conclusión

Node.js es una herramienta fundamental y su instalación es sencilla. Usa nvm si quieres flexibilidad o el instalador directo si prefieres la simplicidad. Con Node funcionando ya estás listo para empezar a construir.
